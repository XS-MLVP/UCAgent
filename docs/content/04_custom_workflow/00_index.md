# 自定义工作流完整指南

欢迎来到 UCAgent 自定义工作流完整教程！本教程将帮助您从零开始掌握 UCAgent 的定制能力，学会构建属于自己的验证工作流。

## 教程目标

通过本教程，您将学会：

- **理解 UCAgent 架构**：掌握 UCAgent 的核心组件和工作原理
- **配置工作流**：学会编写工作流配置文件，定义验证任务的步骤和检查点
- **创建模板文件**：编写模板文件指导 Agent 生成规范文档
- **开发自定义工具**：实现领域专用的工具扩展 UCAgent 能力
- **编写检查器**：开发检查器验证特定业务规则和输出质量
- **端到端实践**：完整构建一个可运行的验证工作流

## 适用读者

本教程适合以下读者：

- **硬件验证工程师**：希望使用 AI 辅助芯片验证的工程师
- **自动化开发者**：需要构建基于 LLM 的自动化工作流的开发者
- **UCAgent 用户**：已经使用 UCAgent 但希望深入定制的用户

**前置要求**：

- 熟悉硬件验证基本概念（对于硬件验证场景）
- 了解 Python 基础语法
- 对 YAML 配置文件有基本了解

## 学习路径

本教程采用"由浅入深、实践先行"的教学方式：

### 📚 第一部分：基础知识（第1章）

了解 UCAgent 的核心架构和工作原理，为后续实践打下基础。

### 🛠️ 第二部分：核心组件（第2-5章）

通过**通用场景的 mini-example**（计算器项目文档生成器）学习四大核心组件：

- 工作流配置（Workflow）
- 模板文件（Templates）
- 自定义工具（Tools）
- 自定义检查器（Checkers）

### 🎯 第三部分：完整实践（第6-8章）

- 整合 mini-example 的完整代码和运行演示
- 深入分析专业领域案例（Formal 形式化验证）
- 掌握运行、调试和问题排查技巧

## Mini-Example 简介

为了降低学习难度，本教程使用**计算器项目文档生成器**作为贯穿示例：

- **场景描述**：为一个计算器应用自动生成项目文档
- **工作流程**：分析项目功能 → 生成规范文档
- **技术栈**：通用文档处理，不涉及复杂的硬件验证知识
- **代码位置**：`examples/MiniWorkflow/`
- **执行时间**：1-10 分钟

所有代码都经过实际测试验证，可以直接运行。

## 如何使用本教程

1. **顺序阅读**：建议按章节顺序学习，每章都基于前面的知识
2. **动手实践**：跟随教程编写代码，运行示例验证理解
3. **对比学习**：先理解 mini-example，再看 Formal 案例的专业应用
4. **查阅文档**：遇到不清楚的概念，通过链接查阅详细文档
5. **迭代改进**：在自己的项目中应用，遇到问题回来查阅
