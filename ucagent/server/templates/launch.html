Manage Launch Files

Tag at least one file as Main RTL or add one .f file, then tag one file as Verification Needs to unlock the next step.
Each file has one primary tag. Only one row can be tagged as Main RTL; assigning a new main file automatically demotes the previous one to Extra RTL. Server-selected .f files stay in place and are passed to picker by absolute path.
Batch mode uses only server-selected YAML launch configs. Upload is disabled; picker and launch settings below apply to every queued task.
