
### Mux 多路选择器

4选1多路选择器，根据选择信号sel选择对应的输入信号输出。

输入：
- in_data[3:0]：4个输入信号（in_data[0], in_data[1], in_data[2], in_data[3]）
- sel[1:0]：2位选择信号

输出：
- out：选择的输出信号

### 验证目标

主要验证多路选择器的选择功能：
1. 验证不同选择信号下的正确输出
2. 验证所有输入通道的连通性
3. 验证默认情况下的行为
4. 验证边界条件

不需要验证其他功能，例如波形、接口时序等。
