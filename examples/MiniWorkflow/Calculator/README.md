# Calculator - 智能计算器应用

## 项目简介

Calculator 是一个功能完善的智能计算器应用，旨在为用户提供便捷、准确的数学计算体验。

## 核心功能

### 1. 基础运算
- **四则运算**：支持加法、减法、乘法、除法运算
- **括号支持**：支持复杂表达式的括号嵌套计算
- **小数计算**：支持高精度小数运算

### 2. 科学计算（扩展功能）
- **三角函数**：sin、cos、tan 及其反函数
- **对数函数**：自然对数 ln、常用对数 log
- **幂运算**：平方、立方、任意次幂计算
- **开方运算**：平方根、立方根计算

### 3. 历史记录
- **计算历史**：自动保存最近100条计算记录
- **历史查询**：支持按日期、表达式查询历史记录
- **结果复用**：可以直接引用历史计算结果

### 4. 错误处理
- **表达式验证**：实时检测表达式语法错误
- **除零检测**：除数为零时给出友好提示
- **溢出保护**：结果超出范围时进行科学计数法转换
- **智能纠错**：对常见输入错误给出修正建议

## 技术特性

- **高精度计算引擎**：采用多精度算法，确保计算准确性
- **跨平台支持**：支持桌面端（Windows、macOS、Linux）和移动端（iOS、Android）
- **轻量级设计**：核心代码不超过2000行，启动速度快
- **可扩展架构**：插件化设计，方便添加新的计算功能

## 使用场景

- 日常生活中的快速计算
- 学生作业和考试中的辅助工具
- 工程技术人员的专业计算
- 金融从业者的财务计算
