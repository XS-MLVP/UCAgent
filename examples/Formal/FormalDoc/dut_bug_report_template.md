# {DUT} 形式化验证缺陷报告

本报告记录了在形式化验证过程中发现的所有 RTL 缺陷。每个缺陷都经过了反例（CEX）分析，并确认不是由于验证环境（约束过紧或过松）导致的。

---

## Failed Property: `A_CK_EXAMPLE_PROP`
- **Checklist Item:** `CK_EXAMPLE_TAG`
- **问题描述**：简要描述在什么条件下触发了错误。
- **根本原因**：分析 RTL 代码中的逻辑错误，给出具体文件和行号。
- **预期行为**：描述设计在协议或规格书要求下的正确表现。
- **实际行为**：描述当前 RTL 的错误表现。
- **修复建议**：提供具体的逻辑修改建议。
